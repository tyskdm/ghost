1. Project management

   - Project

   - Issue
     - Github の課題チケットのことではない。
     - Activity ではなく、その Activitiy が解決しようとしている課題を管理する。
     - 課題タイプごとに、ことなる活動フローをもつ。
       - この活動フローを、プロセスとして定義する。
     - Activity, Task
     - Epic

   - Project management
     - 課題を解決するためのアクティビティの連続。
     - PMは、課題をオーガナイズする。
     - 5つのフェーズをもつ（Initiate, planning,...）

   - Milestone
     - Epic
     - Milestone scheduling
